## What it is

Case-opener is an HTML-driven browser game built around opening cases in a Counter Strike-style format. Since it is made for the web, you can launch it straight in a browser without relying on a dedicated app or launcher.

Its core appeal comes from the motion controls in the opening sequence. By changing speed and friction, you can influence how the animation behaves, which gives you a more adjustable experience for different play preferences or display needs.
